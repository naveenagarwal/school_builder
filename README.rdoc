== README

SchoolBuilder app to show the performance of the test results using thr nvd3 graphs

Things you may want to cover:

* Ruby version 2.1.0

* Rails version 4.0.2

* Create a database.yml file in the app config dorectory (This app is developed using mysql)

* Run bundle install

* Run rake db:create db:migrate db:seed

* Start the server and visit the localhost:3000
